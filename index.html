Equivalency Application Database

Manual input, PDF document storage, database table, edit/delete controls, and Word reply-letter generator.

Important: This version works without a server and can run on GitHub Pages. Data and uploaded PDFs are saved only in the browser/device where you use this website. For shared office use, you will need a backend database such as Firebase, Supabase, or Google Sheets API.

Add New Application

You can upload one or more PDF files. When editing, choose new files only if you want to add more documents.

Backup Tools

Export saves applicant data only. PDFs are stored in this browser database.
